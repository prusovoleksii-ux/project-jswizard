# Mebleria  
## _A Modern Furniture Store Template_

[![JavaScript](https://img.shields.io/badge/JavaScript-ES6-F7DF1E?logo=javascript&logoColor=black)](https://developer.mozilla.org/en-US/docs/Web/JavaScript) [![Vite](https://img.shields.io/badge/Vite-5-purple?logo=vite&logoColor=white)](https://vitejs.dev/) [![Axios](https://img.shields.io/badge/Axios-1.6.0-blue?logo=axios)](https://axios-http.com/) [![HTML](https://img.shields.io/badge/HTML-5-E44D26?logo=html5&logoColor=white)](https://developer.mozilla.org/en-US/docs/Web/HTML) [![CSS](https://img.shields.io/badge/CSS-3-264de4?logo=css3&logoColor=white)](https://developer.mozilla.org/en-US/docs/Web/CSS) [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Mebleria is a sleek, responsive **furniture store web template** powered by JavaScript, Vite, and Axios.  
It’s designed for developers looking to build elegant e-commerce interfaces quickly, featuring interactive components like accordions, dynamic product listings, and a clean modern layout.

---

## Features

- **Dynamic Product Catalog** — Fetch and display products using Axios.  
- **Interactive Accordion UI** — Expandable sections for categories, FAQs, and product descriptions.  
- **Responsive Design** — Polished layout adapting to all device sizes.  
- **Fast Development with Vite** — Lightning build times and instant hot module replacement.  
- **Modular Components** — Organized in reusable HTML/JS blocks for easy scaling.  
- **Custom Styling** — CSS variables for color themes and flexible customization.  
- **npm Integration** — Simple package management for dependencies and scripts.  

---

## Tech

Mebleria is built with:

- [JavaScript (ES6)](https://developer.mozilla.org/en-US/docs/Web/JavaScript) — Dynamic client-side logic.  
- [Vite](https://vitejs.dev/) — Next-generation front-end tooling.  
- [Axios](https://axios-http.com/) — Simplified HTTP requests for APIs.  
- [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML) — Semantic page structure.  
- [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS) — Styling and layout.  
- **Accordion Component** — Custom JS and CSS accordion implementation for UI interactions.
